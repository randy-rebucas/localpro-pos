# Transactions & POS

Complete guide to the Point of Sale system and transaction management.

## Overview

The POS system is the core of 1POS, allowing you to process sales, handle payments, and manage transactions.

## Features

- ✅ Shopping cart system
- ✅ Multiple payment methods (Cash, Card, Digital)
- ✅ Automatic change calculation
- ✅ Receipt generation
- ✅ Transaction history
- ✅ Refund processing
- ✅ Discount application
- ✅ Transaction search and filtering

## Quick Start

### Processing a Sale
1. Go to POS page
2. Search and add products to cart
3. Review cart and totals
4. Apply discount (optional)
5. Select payment method
6. Process payment
7. Print receipt (optional)

### Processing a Refund
1. Go to Transactions
2. Find the transaction
3. Click "Refund"
4. Select items to refund
5. Enter reason
6. Process refund

## Common Tasks

- **New Sale** - POS → Add products → Process payment
- **View Transactions** - Transactions → Browse history
- **Refund** - Transactions → Select transaction → Refund
- **Search Transactions** - Transactions → Use search/filters
- **Print Receipt** - Transaction details → Print
