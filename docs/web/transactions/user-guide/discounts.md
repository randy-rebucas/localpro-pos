# Applying Discounts - User Guide

Complete guide to applying discount codes and promotions in transactions.

## Overview

Discounts can be applied to transactions to reduce the total amount. Discounts can be percentage-based or fixed amounts, and may have conditions like minimum purchase requirements.

## Accessing Discount Feature

### From POS Interface

1. Add products to cart
2. Review cart totals
3. Click **"Apply Discount"** button
4. Discount dialog appears
