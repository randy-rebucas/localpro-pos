# Processing Refunds - User Guide

Complete guide to processing refunds and returns in 1POS.

## Overview

The refund system allows you to process full or partial refunds for transactions, with automatic stock restoration and proper transaction tracking.

## Accessing Refund Feature

### From Transactions Page

1. Navigate to **Transactions**
2. Find the transaction to refund
3. Click on transaction
4. Click **"Refund"** button

### From Transaction Details

1. View transaction details
2. Click **"Process Refund"** button
3. Refund dialog appears
