# Processing Sales - Step by Step Guide

Complete step-by-step guide to processing sales in the POS system.

## Overview

This guide walks you through the complete sales process from adding products to completing the transaction.

## Step-by-Step Process

### Step 1: Start New Sale

1. Navigate to **POS** page
2. Ensure cart is empty (or clear existing items)
3. You're ready to start a new sale

### Step 6: Select Payment Method

Choose appropriate payment method:

#### Cash Payment

1. Click "Cash" button
2. Enter amount received from customer
3. System calculates change automatically
4. Verify change amount
5. Click "Process Payment"

- Change: $6.16

#### Card Payment

1. Click "Card" button
2. Process card through card reader
3. Wait for approval
4. Click "Process Payment" when approved

## Complete Example

### Example Sale: Coffee Shop

**Customer Order**:
- 2x Large Coffee ($5.99 each)
- 1x Muffin ($3.50)
- 1x Cookie ($2.00)

**Process**:

1. **Add Items**:
   - Click "Large Coffee" twice (quantity: 2)
   - Click "Muffin" once
   - Click "Cookie" once

2. **Review Cart**:
   - Large Coffee x2: $11.98
   - Muffin x1: $3.50
   - Cookie x1: $2.00
   - Subtotal: $17.48

3. **Apply Discount**:
   - Customer has "COFFEE10" code
   - Apply discount: -$1.75 (10% off)
   - New Total: $15.73

4. **Process Payment**:
   - Customer pays with card
   - Select "Card" payment
   - Process card
   - Complete transaction

5. **Receipt**:
   - Receipt number: REC-20240101-00042
   - Print receipt
   - Give to customer

## Common Patterns

### Pattern 1: Single Item Sale

1. Click product
2. Select payment method
3. Complete transaction
4. Print receipt

**Time**: ~10 seconds

### Pattern 2: Multiple Items Sale

1. Add multiple products
2. Review cart
3. Apply discount (if any)
4. Process payment
5. Print receipt

**Time**: ~30-60 seconds

### Pattern 3: Sale with Quantity Changes

1. Add products
2. Adjust quantities
3. Review totals
4. Process payment
5. Print receipt

**Time**: ~45 seconds

## Tips for Speed

### Keyboard Shortcuts

- **Space** - Quick payment (when ready)
- **Ctrl/Cmd + F** - Focus search
- **Esc** - Cancel action
- **Enter** - Confirm action

### Workflow Optimization

1. **Keep Scanner Ready** - For barcode scanning
2. **Know Product Locations** - Faster product selection
3. **Use Search** - Faster than scrolling
4. **Batch Similar Items** - Add all of one type at once
5. **Prepare Change** - For cash, have change ready

## Error Handling

### Insufficient Stock

**Problem**: Product out of stock

**Solution**:
1. System shows "Out of Stock" message
2. Inform customer
3. Suggest alternative product
4. Or remove from cart

> **Screenshot Placeholder**: [Add screenshot of out of stock message]

### Invalid Discount

**Problem**: Discount code not valid

**Solution**:
1. System shows error message
2. Verify code with customer
3. Try again or proceed without discount

### Payment Failure

**Problem**: Payment processing fails

**Solution**:
1. Check error message
2. Verify payment method
3. Try again
4. Or use different payment method
