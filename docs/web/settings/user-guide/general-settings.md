# General Settings - User Guide

Complete guide to configuring general system settings in LocalPro POS.

## Overview

General settings allow you to configure basic system preferences, including currency, language, date formats, and display options.

## Accessing Settings

### From Main Menu

1. Navigate to **Settings**
2. General settings tab appears
3. Configure options

## Currency Settings

### Setting Currency

1. Go to Currency section
2. Select currency from dropdown
3. Options include:
   - USD ($)
   - EUR (€)
   - GBP (£)
   - And more...
4. Save settings

### Currency Display

Configure how currency displays:
- **Symbol Position**: Before or after amount
- **Decimal Places**: Number of decimals
- **Thousands Separator**: Comma, period, or space
- **Decimal Separator**: Comma or period
