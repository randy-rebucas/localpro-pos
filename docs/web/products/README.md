# Products Management

Complete guide to managing products in the LocalPro POS web application.

## Overview

The Products module allows you to manage your product catalog, including variations, bundles, inventory, and pricing.

## Features

- ✅ Product CRUD operations
- ✅ Product categories
- ✅ SKU tracking
- ✅ Product variations (size, color, type)
- ✅ Product bundles
- ✅ Inventory management
- ✅ Barcode/QR code support
- ✅ Product images
- ✅ Search and filtering

## Quick Start

### Adding a Product
1. Go to Admin → Products
2. Click "Add Product"
3. Fill in product details (name, price, stock)
4. Select category
5. Add variations if needed
6. Save

### Creating Variations
1. Edit product
2. Go to Variations section
3. Add variation (size, color, etc.)
4. Set price and stock for each variation
5. Save

## Common Tasks

- **Add Product** - Admin → Products → Add Product
- **Edit Product** - Admin → Products → Click product → Edit
- **Manage Stock** - Admin → Products → Click product → Refill Stock
- **Create Bundle** - Admin → Bundles → Add Bundle
- **Search Products** - Use search bar in Products page
