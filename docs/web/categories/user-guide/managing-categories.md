# Managing Categories - User Guide

Complete guide to creating and managing product categories in 1POS.

## Overview

Categories help organize your products, making them easier to find in the POS and manage in the admin panel.

## Accessing Categories

### From Products Page

1. Go to **Admin** → **Products**
2. Click **"Categories"** tab
3. Category management appears

## Best Practices

### Category Organization

- ✅ Use clear, descriptive names
- ✅ Keep categories focused
- ✅ Don't create too many categories
- ✅ Review category structure regularly

### Product Assignment

- ✅ Assign all products to categories
- ✅ Use consistent assignment
- ✅ Review uncategorized products
- ✅ Update as product mix changes
