# 20. Daily Operations Checklist

## Opening Checklist (Start of Day)

### Cashier

- [ ] Log in to 1POS (email/password, PIN, or QR badge)
- [ ] Clock in via the Attendance Clock
- [ ] Open the Cash Drawer with the correct opening balance
- [ ] Verify the opening cash matches the physical count
- [ ] Check for any saved/held carts from previous shift
- [ ] Confirm barcode scanner and receipt printer are working
- [ ] Process test receipt if needed (void after)

### Manager

- [ ] Complete all Cashier opening steps
- [ ] Review low stock alerts on the dashboard
- [ ] Check today's bookings/appointments
- [ ] Review any pending stock transfers
- [ ] Check staff attendance — ensure all scheduled staff are clocked in
- [ ] Review any overnight system notifications

### Admin / Owner

- [ ] Complete all Manager opening steps
- [ ] Check subscription status and usage limits
- [ ] Review audit logs for any overnight activity
- [ ] Verify automated backup completed successfully
- [ ] Check for any system alerts or errors

---

## During the Day

### Every Sale
- [ ] Scan or search for products
- [ ] Verify quantities and prices
- [ ] Apply discounts if applicable
- [ ] Confirm payment method and amount
- [ ] Provide receipt (print or email)

### Hourly
- [ ] Glance at the offline indicator — ensure you're connected
- [ ] Check for any pending saved carts

### Midday (Manager)
- [ ] Quick inventory check on fast-moving items
- [ ] Review morning sales numbers
- [ ] Check afternoon bookings

---

## Closing Checklist (End of Day)

### Cashier

- [ ] Complete all pending transactions
- [ ] Check for any saved/held carts — process or clear them
- [ ] Count physical cash in the drawer
- [ ] Close the Cash Drawer:
  - Enter actual cash count
  - Note any discrepancy with explanation
- [ ] Clock out via the Attendance Clock
- [ ] Log out of 1POS

### Manager

- [ ] Ensure all cashiers have closed their drawers
- [ ] Review daily sales summary on the Reports page
- [ ] Check cash drawer discrepancy reports
- [ ] Review and approve any pending refunds
- [ ] Export the Sales Journal for the day (CSV or PDF)
- [ ] Check stock levels — flag anything for restock
- [ ] Review tomorrow's bookings
- [ ] Clock out
- [ ] Log out

### Admin / Owner

- [ ] Review all Manager closing items
- [ ] Export VAT report if end of reporting period
- [ ] Verify daily backup completed
- [ ] Review audit logs for the day
- [ ] Check subscription usage (transactions, products)
- [ ] Address any system alerts

---

## Weekly Tasks

### Manager
- [ ] Full inventory count for high-value items
- [ ] Review product performance report
- [ ] Process pending stock transfers
- [ ] Review staff attendance summary
- [ ] Plan restocking for the coming week

### Admin / Owner
- [ ] Review weekly sales and P&L report
- [ ] Check user accounts — deactivate any that are no longer needed
- [ ] Review and clean up discount codes
- [ ] Verify backup integrity
- [ ] Review system audit logs for unusual activity

---

## Monthly Tasks

### Manager
- [ ] Complete physical inventory count (all items)
- [ ] Reconcile stock adjustments
- [ ] Review and update product prices if needed
- [ ] Submit expense reports

### Admin / Owner
- [ ] Generate monthly VAT report (for BIR Form 2550M)
- [ ] Generate monthly P&L report
- [ ] Review subscription usage and plan adequacy
- [ ] Review and update tax rules if changed
- [ ] Verify 10-year data retention / archiving is running
- [ ] Update business hours for next month if needed
- [ ] Review and update holiday calendar

---

## Quarterly Tasks (Admin / Owner)

- [ ] Generate quarterly VAT report (for BIR Form 2550Q)
- [ ] Full system backup and verification
- [ ] Review user roles and access — remove unnecessary permissions
- [ ] Review automation configurations
- [ ] Test disaster recovery — can you restore from backup?
- [ ] Review hardware status — printers, scanners

---

## Tips

1. **Print this checklist** and keep it near the register
2. **Initial each step** as you complete it
3. **Report issues immediately** — don't wait until end of day
4. **Document discrepancies** — always add notes when cash doesn't balance
5. **Never skip the closing process** — proper closing prevents morning surprises
