# 2. Dashboard Overview

The dashboard is the first screen after login. It provides a real-time snapshot of your store's performance.

## Dashboard Widgets

### Sales Summary

- **Today's Sales** — Total revenue for the current day
- **This Week** — Weekly sales total
- **This Month** — Monthly sales total
- **Sales Chart** — Visual graph showing sales trends over time

### Quick Stats

- **Total Transactions** — Number of completed sales today
- **Average Order Value** — Mean transaction amount
- **Items Sold** — Total quantity of products sold
- **Active Customers** — Unique customers served today

### Inventory Alerts

- **Low Stock Items** — Products below their reorder threshold
- **Out of Stock** — Products with zero inventory
- Click any alert to go directly to the inventory page

### Recent Transactions

- Shows the last 5-10 transactions
- Displays transaction ID, amount, payment method, and time
- Click any transaction to view full details

## Refreshing Data

- Dashboard data updates automatically via real-time events
- To force refresh, use the browser refresh button or pull down on mobile
- Sales data reflects only the current branch unless you have multi-branch access
