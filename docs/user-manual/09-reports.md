# 9. Reports & Analytics

**Available to:** Manager, Admin, Owner

## Accessing Reports

1. Navigate to **Reports** from the sidebar
2. Select the report type from the tabs

## Available Reports

### Sales Report

Comprehensive sales overview with filtering:

| Metric | Description |
|--------|-------------|
| **Gross Sales** | Total revenue before discounts |
| **Net Sales** | Revenue after discounts and refunds |
| **Discount Total** | Total discounts given |
| **Refund Total** | Total refunds issued |
| **Transaction Count** | Number of sales |
| **Average Order Value** | Mean sale amount |

**Filters:**
- Date range (Today, This Week, This Month, Custom)
- Branch
- Cashier
- Payment method

**Chart:** Visual sales trend line by day, week, or month.

### Product Performance Report

| Metric | Description |
|--------|-------------|
| **Top Sellers** | Products ranked by units sold |
| **Revenue by Product** | Products ranked by revenue generated |
| **Low Performers** | Products with fewest sales |
| **Category Breakdown** | Sales distribution by category |

### VAT Report

BIR-compliant tax report:

| Field | Description |
|-------|-------------|
| **VATable Sales** | Total sales subject to 12% VAT |
| **VAT Amount** | 12% of VATable sales |
| **VAT-Exempt Sales** | Sales exempt from VAT |
| **Zero-Rated Sales** | Sales at 0% VAT rate |
| **Total Sales** | Grand total |

### Profit & Loss (P&L) Report

| Section | Description |
|---------|-------------|
| **Revenue** | Total net sales |
| **Cost of Goods Sold** | Total cost of products sold |
| **Gross Profit** | Revenue minus COGS |
| **Expenses** | Business expenses entered in the Expenses module |
| **Net Profit** | Gross profit minus expenses |

### Cash Drawer Report

| Metric | Description |
|--------|-------------|
| **Opening Balance** | Cash at start of session |
| **Cash Sales** | Cash received from sales |
| **Cash Refunds** | Cash given for refunds |
| **Expected Balance** | Calculated amount that should be in drawer |
| **Actual Count** | Entered by cashier at closing |
| **Discrepancy** | Difference (overage or shortage) |

### Sales Journal

Detailed line-by-line record of every transaction:

| Column | Description |
|--------|-------------|
| **Date/Time** | Transaction timestamp |
| **Receipt #** | Serial receipt number |
| **Customer** | Customer name (if linked) |
| **Items** | Product names and quantities |
| **Subtotal** | Pre-tax amount |
| **VAT** | Tax amount |
| **Discount** | Discount applied |
| **Total** | Final amount |
| **Payment** | Payment method |
| **Cashier** | Staff who processed |

### Bundle Performance

For stores using product bundles:
- Bundle sales count
- Revenue per bundle
- Component product tracking
- Comparison to individual product sales

## Exporting Reports

All reports can be exported:

1. Apply your desired filters
2. Click **Export**
3. Choose format:

| Format | Best For |
|--------|----------|
| **CSV** | Spreadsheet analysis, importing to other systems |
| **Excel** | Formatted reports with formatting preserved |
| **PDF** | Printing, archiving, sharing with accountant |

4. The file downloads to your device

## Daily Sales Summary (Z-Reading Equivalent)

For BIR compliance, the Daily Sales Summary serves as the Z-reading:

1. Navigate to **Reports > Sales Report**
2. Set date to **Today**
3. The summary shows all required BIR fields
4. Export as PDF for your records

## Scheduled Reports

If automation is enabled, reports can be emailed automatically:
- Daily sales summary sent each night
- Weekly performance report sent Monday morning
- Monthly P&L sent on the 1st of each month

Configure in **Settings > Automations > Report Delivery**.
