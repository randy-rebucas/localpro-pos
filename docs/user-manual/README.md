## User Roles

1POS has five user roles, each with progressively more access:

| Role | Level | Description |
|------|-------|-------------|
| **Viewer** | 1 | Read-only access to dashboard, products, and transactions |
| **Cashier** | 2 | Process sales, refunds, manage cash drawer, clock in/out |
| **Manager** | 3 | Everything Cashier can do + products, inventory, reports, staff |
| **Admin** | 4 | Everything Manager can do + settings, users, audit logs, tax config |
| **Owner** | 5 | Full system access + tenant management, subscriptions, billing |

Start with [Getting Started](./01-getting-started.md) to learn how to log in and navigate the system.
